# RISE-Internship_Shweta_TS-RISE-CEH-2531307_Password-Strength-Checker
A simple Python tool that checks how strong a password is based on regex rules.
It analyzes length, uppercase/lowercase letters, digits, and special characters, then labels the password as Weak, Medium, or Strong.

# Tech Used

-> Python
-> Tkinter
-> Regex

# Run
python password_strength_checker.py

# Features

-> Easy GUI
-> Instant strength evaluation
